# Benoit

- **Description** - Benoit is a programming language that transpiles to JavaScript, optimized for human-AI pair programming. It achieves ~68% token reduction versus equivalent JavaScript, making it ideal for LLM-assisted coding. Its most distinctive feature is inline test assertions as first-class syntax — tests live right next to the function definition and are automatically extracted into a test harness. The entire transpiler is ~500 lines with zero dependencies.
